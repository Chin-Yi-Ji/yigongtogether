# 新建檔案（Onboarding）流程示範

## 整體流程

新建檔案分 **三步**，依序為：**目前位置** → **興趣關注** → **自我介紹與學經歷**。

---

## Step 1：目前位置

一個卡片，標題「目前位置」，副標「選擇你的所在地區（之後可再編輯）」。

下方是一排圓角按鈕（pill）：

```
[ 北部 ]  [ 中部 ]  [ 南部 ]  [ 東部 ]  [ 國外 ]
```

選「國外」時會多一個輸入框：「自訂位置（hashtag）」，placeholder 例如：東京、新加坡、矽谷。

最下方一個主按鈕：「下一步：興趣關注」。

---

## Step 2：興趣關注（四向度、依人數排序）

一個卡片，標題「興趣關注」，副標「點選你有興趣的標籤，最多選 5 個（已選 0/5）」。

**標籤依四向度分組顯示，畫面上不顯示向度名稱**，只是由上到下分成四「區塊」：

- **第一區塊**（動態興趣，組內依使用人數由多到少）  
  例：`#旅行` `#運動` `#音樂` `#攝影` `#自由潛水` `#內容創作` `#社群經營`

- **第二區塊**（靜態興趣，組內依人數排序）  
  例：`#閱讀` `#設計` `#寫作` `#開源` `#永續` `#公益` `#環保`

- **第三區塊**（專業能力，組內依人數排序）  
  例：`#AI` `#前端` `#產品` `#行銷` `#UX` `#創業` … 

- **第四區塊**（其他 + DB 有但未在預設分類的標籤）  
  例：`#社會創新` 以及任何資料庫裡有、但不在上面三類的標籤

每個標籤是圓角按鈕，未選為次要樣式，點選後變為主色並有 ring。最多選 5 個。

下方：「上一步」按鈕 + 「下一步：自我介紹」主按鈕。

---

## Step 3：自我介紹與學經歷

多張卡片依序排列：

1. **頭貼**：上傳大頭照（選填），或使用登入帳號頭像。
2. **基本資料與自我介紹**  
   - 個人連結名稱、顯示名稱、目前位置（唯讀，來自 Step 1）  
   - 聯絡方式、四省中科博館義工（屆數、義工之友）  
   - 學歷（可多筆，每筆可新增/刪除）  
   - 工作經歷（可多筆，「還是學生」勾選則工作經歷選填）  
   - 興趣：顯示 Step 2 已選 + 可再輸入額外 hashtag（空格或逗號分隔）  
   - 自我介紹 *（500 字以內）

最下方：「上一步」+ 「完成建立」。

---

## 如何實際看到畫面

在專案目錄執行：

```bash
npm run dev
```

瀏覽器開啟 **http://localhost:3000/onboarding**（需先登入，未登入會導向登入頁）。  
若已有 Profile 會導向 `/u/{handle}`，可改用無 Profile 的帳號或先登出再訪問 `/onboarding`。

即可依序看到 Step 1 → Step 2（四區塊標籤、不顯示向度名稱）→ Step 3 的實際呈現。
